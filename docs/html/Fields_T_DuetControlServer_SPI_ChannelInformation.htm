ChannelInformation Fields

[This is preliminary documentation and is subject to change.]

The ChannelInformation type exposes the following members.

Fields
  NameDescription
BufferedCodes
List of buffered G/M/T-codes that are being processed by the firmware
NestedMacroCodes
Pending codes being started by a nested macro (and multiple codes may be started by an interceptor). This is required because it may take a moment until they are internally processed
NestedMacros
Stack of nested macro files being executed
PendingCodes
Queue of pending G/M/T-codes that have not been buffered yet
PendingFlushRequests
Queue of pending flush requests
PendingLockRequests
Queue of pending lock/unlock requests
SuspendedCodes
Stack of suspended G/M/T-codes to resend when the current macro file finishes
SystemMacroHasFinished
Indicates whether the requested system macro file has finished
